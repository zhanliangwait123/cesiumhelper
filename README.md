产品名称：[ Cesium 助手 ] —— 全场景三维数据一键切片与发布平台
在实景三维中国、数字孪生城市与Cesium应用日益普及的今天，如何高效地将多源异构数据转换为标准的3D Tiles，并快速发布服务，成为项目落地的关键瓶颈。Geo3D-Publisher Pro 正是为解决这一痛点而生的专业工具。

核心功能介绍
本产品提供从原始数据到最终服务发布的一站式解决方案，全面覆盖以下核心能力：

1. 全要素数据转3D Tiles切片
支持将多种格式的原始数据无损、高效地转换为Cesium原生支持的3D Tiles格式：

地形数据：支持DEM、GeoTIFF等生成地形切片。

影像数据：支持DOM、正射影像生成影像图层。

倾斜摄影：支持OSGB格式的倾斜摄影模型直接转为3D Tiles。

人工模型：支持FBX、OBJ、DAE、IFC（ BIM模型）等精细手工模型。

矢量数据：支持Shapefile、GeoJSON、KML、DWG等海量矢量数据转换为矢量3D Tiles。

2. 多坐标系深度兼容
完美适配Cesium的多种坐标体系，无需手动进行复杂的投影转换：

球面坐标：完整支持 WGS84 地理坐标系。

平面坐标：全面支持Cesium常用的各类平面坐标系（含Web Mercator及任意地方独立坐标系），满足地下管线、园区级高精度平面场景需求。

3. 矢量瓦片支持
除了3D Tiles，同时支持生成 MapBox Vector Tiles (MVT) 格式的矢量瓦片，实现矢量数据的轻量化动态加载与样式实时切换。

4. 一键数据发布
内置轻量化服务引擎，切片完成后支持：

一键发布本地3D Tiles服务与WMTS/TMS服务。

自动生成服务地址（URL），可直接复制到Cesium中预览。

支持局域网内其他终端访问，实现即切即用。

5. 坐标轴编辑
针对不同建模软件（如3ds Max、Revit）轴向定义差异，内置可视化坐标轴编辑工具：

支持平移、旋转、缩放坐标原点。

可手动调整模型姿态（绕X/Y/Z轴旋转），解决模型“倒地”或“位置偏移”问题，确保数据精准匹配底图。

适用场景
智慧城市：倾斜摄影+建筑物精模+LOS数据一体化发布。

数字孪生工厂：BIM模型坐标轴校正与轻量化展示。

军事/应急仿真：地形与影像快速生成高程服务。

WebGIS开发：为Cesium开发者提供开箱即用的数据服务接口。
